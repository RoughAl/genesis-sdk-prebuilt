This directory contains redistributions of binaries and source code
from the 6-2017-q2-update (June 28, 2017) release of the GNU ARM
Embedded Toolchain originally hosted here:

https://developer.arm.com/open-source/gnu-toolchain/gnu-rm/downloads

The binary releases for Linux and macOS targets are respectively
included in "linux" and "mac" subdirectories.  For the purposes of GPL
compliance, the Corresponding Source used to generate these binary
releases can be found in the "src" subdirectory.  These subdirectories
contain the results of extracting binary release archives with the
following checksums.
